"# Selenium-Framework" 

This is a Selenium Framework for automated testing of web applications. It provides a structured approach to writing and executing test cases using Selenium WebDriver. The framework includes features such as test case management, reporting, and support for multiple browsers. It is designed to be easy to use and maintain, allowing testers to focus on writing effective test cases rather than dealing with the complexities of the underlying automation framework.

Prerequisites:

Java 17 or higher

Maven 3.9.6
